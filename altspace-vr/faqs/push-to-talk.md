# <a name="is-there-a-push-to-talk-button"></a>PTT(Push-to-Talk) 단추가 있나요?

보기의 오른쪽 아래에는 음성이 다른 사용자에게 공유되는지 여부를 토글하는 데 사용할 수 있는 마이크 단추가 있습니다. 또는 바로 가기 키 **Ctrl + Alt + T를** 사용하여 이 설정을 전환할 수 있습니다. 
 
대화할 때 아이콘이 깜박이면 마이크가 작동합니다.
 
질문이나 의견이 있으신가요?
