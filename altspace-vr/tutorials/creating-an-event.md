# <a name="creating-an-event"></a>이벤트 만들기

AltspaceVR에서 이벤트를 만드는 방법에 대 한 단계별 가이드입니다. AltspaceVR의 여러 이벤트에 참석 하 여 작동 방식에 대 한 느낌을 얻는 것이 좋습니다. [AltspaceVR Events 달력](https://account.altvr.com/events) 에서 모든 이벤트 목록을 확인 합니다.

이 문서에서는 다음에 대해 알아봅니다.

* AltspaceVR 이벤트 용어
* 이벤트 및 이벤트 공간 만들기
* 슬라이드 프레젠테이션에 멀티미디어 콘솔 사용
* 이미지를 사용 하 여 이벤트 브랜딩
* 세계 편집기/키트를 사용 하 여 3D 개체 추가
* 내 이벤트를 기록 하거나 라이브 스트림 하는 방법

## <a name="altspacevr-event-terminology"></a>AltspaceVR 이벤트 용어

이벤트 및 이벤트 공간을 만들기 위해 알아야 하는 용어는 다음과 같습니다.

</br>

| 용어 | 정의 |
|---|---|
| 세계 빌드 | AltspaceVR는 가상 환경을 만들고 사용자 지정 하는 기능을 제공 합니다. AltspaceVR 호스트는 설명서, Discord 채널 및 전 세계의 이벤트를 지원 하 여 [도움말을 얻고 빌드를 시작 하는 방법에 대 한 자세한 내용을](../world-building/world-building-faq.md)확인할 수 있습니다. |
| World | 세계는 AltspaceVR의 가상 공간입니다. Office 또는 방대한 양의 산악 것일 수 있습니다. 매우 사용자 지정이 가능한 환경입니다. 이는 Universe 내에 있으며, 해당 Universe 내에 여러 가지가 있을 수 있습니다. 특별 한 이벤트, 학습 센터 및 다른 모임 공간에 대 한 여러 이벤트 공간을 꾸며보세요 하 게 하려면 동일한 Universe에 있는 세계를 추가 하 여 그룹으로 함께 유지 합니다. |
| Universe | AltspaceVR 세계의 universe vernacular 건물은 세계의 범주화를 나타냅니다. 각 universe는 여러 가지를 보유할 수 있습니다. 이 세계는 universe의 설정을 상속 하 여 여러 세계 및 기타 제어 기능에 대 한 허용 목록에 사용자를 쉽게 추가할 수 있도록 합니다. 자세한 내용은 [사용자 환경 관리](../world-building/managing-worlds.md) 를 참조 하세요. |
| 템플릿 | 템플릿 (또는 공간 템플릿)은 전 세계의 기능을 사용 하 여 생성 하는 대신 사용할 수 있는 미리 만들어진 세계 또는 환경입니다. AltspaceVR는 다양 한 환경 및 이벤트에 대 한 다양 한 템플릿을 제공 합니다. |
| 이벤트 공간 | 이벤트 공간은 세계 AltspaceVR의 동의어입니다. 일반적으로 이벤트를 호스트 하는 데 사용 되는 세계를 나타냅니다. |
| 웹 사이트 | "웹 사이트"에 대 한 참조는 [AltspaceVR의 웹 사이트](https://altvr.com/)입니다. VR 장치가 아닌 컴퓨터 또는 태블릿에서 [이벤트 웹 페이지](https://account.altvr.com/events/my) 를 통해 이벤트를 만들고 편집 하는 것이 더 쉬운 경우가 많습니다. 또한 [세계 빌드](../world-building/managing-worlds.md) 를 위해 웹 사이트에 액세스 해야 합니다. |
| 상황별 역할 | [상황별 역할](../getting-started/roles.md) 은 이벤트 작성자 또는 전역 작성기에서 할당 합니다. 이러한 역할은 전 세계의 사용자에 게 추가 기능과 기능을 제공 합니다. 현재 이러한 구성 요소는 호스트, 중재자, 파일럿 (비행), Terraraw (세계 빌딩) 및 폰로 구성 됩니다. 이러한 이벤트는 개별적으로 또는 전역적으로 할당 될 수 있으므로 모든 사용자가 이벤트 공간 또는 세계에서 동일한 역할을 가질 수 있습니다. |
| UI (사용자 인터페이스)/메뉴 | 모던 환경 내에서 AltspaceVR 내에 있는 경우 화면의 왼쪽과 오른쪽에 메뉴가 있습니다. AltspaceVR 로고가 있는 원 또는 주 메뉴는 기본 UI (사용자 인터페이스) 또는 메뉴를 열어 AltspaceVR을 탐색 하 고 사용자 환경을 사용자 지정 하는 다양 한 화면에 액세스 합니다. 선택적 UI 항목은 화면의 오른쪽 크기에 있으며 일반적으로 세계 편집기와 호스트 도구를 포함 합니다. 이러한 항목은 커서를 사용 하 여 열고 상호 작용 합니다. |
| SDK/MRE | 이러한 용어는 세계 구축 환경에 기능과 기능을 추가 하는 데 사용 되는 [혼합 현실 확장과](../world-building/using-mixed-reality-extensions.md) 소프트웨어 개발 키트와 관련 된 용어입니다. 이러한 설정은 일반적으로 고급 사용자를 위한 것입니다. |

## <a name="understanding-events"></a>이벤트 이해

AltspaceVR를 사용 하면 미리 만든 템플릿 공간을 사용 하 여 특정 요구에 맞게 사용 하거나 사용자 지정 하거나 처음부터 새로 만들 수 있는 전 세계를 쉽게 만들 수 있습니다. 이 문서에서는 미리 만든 템플릿을 사용 하 여 이벤트를 만드는 방법을 설명 합니다. 다음 섹션에서는 [이미지를 사용 하 여 이벤트를 브랜딩](#branding-your-event-with-images) 하 고 [세계 편집기 및 키트를 사용 하 여 3D 개체를 추가 하](#adding-3d-objects-using-world-editor-and-kits) 는 방법에 대해 자세히 설명 합니다. 사용자 지정 세계를 처음부터 구축 하는 방법에 대 한 자세한 내용은 AltspaceVR에서 둘러보기 모임을을 빌드하고 [전 세계 지원 설명서를 확인 하세요](../world-building/world-editor-getting-started.md).

AltspaceVR는 이벤트에 사용할 공간을 만드는 두 가지 방법을 제공 합니다.

* **일회성 사용:** 이벤트를 만들고 템플릿 세계를 선택 합니다.
* **반복 사용:** 세계 공간을 만들어 이벤트로 가져옵니다.

이벤트를 만드는 프로세스는 둘 다에 대해 동일 합니다. 단, [전 세계의 생성 및](../world-building/managing-worlds.md) 반복 사용 이벤트 공간으로 가져오기는 예외입니다. 다음 사항도 알고 있어야 합니다.

</br>

| 용어 | 정의 |
|---|---|
| 세계 복사 됨 | 반복 되는 이벤트에 대해 세계 공간을 사용 하면 지속적으로 사용자 지정할 수 있습니다. 기호, 이미지, 생성 시점 및 기타 사용자 지정은 매번 이벤트 공간을 사용자 지정 하는 대신 이벤트에서 이벤트로 유지 관리 됩니다. |
| Event 세계 사용자 지정 | 이벤트가 생성 되 면 템플릿이나 가져온 전 세계는 해당 이벤트로 복사 되 고 잠깁니다. 원래 전 세계에는 영향을 주지 않고 이벤트를 변경 하 고 그 밖의 다른 방법을 사용할 수 있습니다. 명절 장식, 그림 또는 다른 장식용 일회성 편집기 요소를 추가 하 여 이벤트 공간을 보다 즐겁게 그리고 이벤트에 적합 하 게 만드는 것이 좋습니다. |

이벤트 공간이 업데이트 되지 않는 한 원래 전 세계의 수정 내용이 이벤트에 적용 되지 않습니다.
1. 이벤트 웹 페이지에서 **다시 가져오기 전** 단추를 사용 합니다.
2. 동기화가 완료 될 때까지 2-3 분을 허용 합니다. 
3. 이벤트 공간이 변경 되지 않은 경우 **설정 > > 보통** 으로 이동 하 여 공간을 다시 설정 하 여 이벤트 공간을 다시 설정 합니다. 공간을 세계 빌더로 다시 설정 합니다.

제대로 로드 되지 않는 요소와 같은 기술적인 문제가 있는 경우 AltspaceVR 메뉴로 이동 하 여 **설정 > 보통 >** 를 선택 하 여 이벤트 공간을 다시 설정 하 고 새 변경 내용을 확인 합니다. Windows 2D의 PC 사용자는 바로 가기 키 (AltspaceVR의 **CTRL + ALT + R** )를 사용 하 여 공간을 신속 하 게 다시 설정할 수 있습니다.

## <a name="creating-your-event-and-event-space-or-world"></a>이벤트 및 이벤트 공간 만들기

다음은 일회성 또는 반복 이벤트에 대 한 이벤트를 만드는 단계별 지침입니다. 템플릿을 선택 하거나 이벤트에 대 한 전 세계를 가져옵니다. 

> [!NOTE]
> AltspaceVR events 웹 페이지에서는 녹색 물음표 단추를 사용 하 여 양식의 모든 측면에 대 한 지침을 제공 합니다. 특정 명령에 대 한 커서를 롤로 롤포워드합니다.

AltspaceVR의 웹 사이트에 있는 이벤트 [> 내 이벤트](https://account.altvr.com/events/my) 웹 페이지에서 **이벤트 일정** 을 선택 하거나 [AltspaceVR에서 이벤트 만들기 웹 페이지로](https://account.altvr.com/events/new)바로 이동 합니다.

1. **이벤트 제목:** 이벤트 이름을 입력 합니다. AltspaceVR의 웹 사이트 및 내 인터페이스에서 다양 한 달력 보기를 볼 수 있도록 구체적이 고 간결 하 게 유지 합니다. 단어 사이에 공백을 포함 하 여 23 자 아래에 제목을 유지 해 보세요.
2. **설명:** 이벤트에 대 한 설명을 입력 합니다.
    * 설명에 10 자 이상의 문자가 있어야 합니다. 그렇지 않으면 이벤트가 생성 되지 않습니다.
    * 단락 사이에 빈 공백을 추가 합니다.
    * 이벤트의 Facebook, Discord, 웹 사이트 또는 기타 리소스에 대 한 링크를 추가 하려면 다음 형식을 사용 합니다 .이 Markdown는 웹 사이트의 이벤트 수준 올리기 페이지 에서만 작동 합니다 .이는 AltspaceVR 메뉴에서 제대로 렌더링 되지 않습니다. `[Event Name](http://example.com/)`

3. **시작 날짜/종료 날짜:** 시작 시간을 설정 하 고 종료 시간이 시작 시간 이후 인지 확인 합니다.
4. **범주:** 이벤트 유형을 가장 잘 설명 하는 범주를 선택 합니다. 
5. 이벤트를 **Private** 또는 **Public** 으로 설정 합니다.
    * 표시 유형: 공용 이벤트는 AltspaceVR 웹 사이트 이벤트 달력의 [모두 탭](https://account.altvr.com/events/all) 에 표시 되 고 공용으로 열립니다.
    * 전용 이벤트는 AltspaceVR 이벤트 달력에 표시 되지 않으며 이벤트에 대 한 URL을 입력 하려면 이벤트 공간을 입력 해야 합니다.
    * ' 주 이벤트로 추가 ' 하려는 경우 이벤트를 공용으로 설정 해야 합니다.
6. **템플릿 선택:** 웹 페이지의 오른쪽에는 AltspaceVR에서 사용 가능한 템플릿의 미리 보기 이미지 목록이 있습니다. 게임 방, 사무실, 모임 공간, 프레젠테이션 공간 및 재미 있는 모임 공간이 있습니다. 흥미를 보이는 항목을 선택 합니다. 원하지 않는 경우 계속 진행 하 여 새 템플릿으로 새 이벤트를 만듭니다. 사용자 고유의 사용자 지정 이벤트 세계를 만들려면 기본 템플릿으로 **쿨 하늘** 을 선택 하 고 전 세계를 가져오는 다음 지침을 따릅니다.
7. **고급 옵션** 선택

### <a name="advanced-options"></a>고급 옵션

1. **수준 올리기:** 각 이벤트에는 두 개의 브랜딩 이미지가 필요 합니다. 이러한 이미지는 AltspaceVR 웹 사이트에 표시 되며 AltspaceVR 내부의 이벤트에 나타나지 않습니다.
    * **타일:** 타일 이미지는 1920 x 1080 픽셀 이어야 합니다. 이 이미지는 다양 한 축소판 그림 크기에 맞게 크기가 조정 되 고 AltspaceVR events 달력에서 다른 이벤트와 함께 표시 됩니다. 이미지가 명확 하 고 많은 텍스트를 포함 하지 않는지 확인 합니다. 소유 하지 않은 저작권 이미지/로고는 사용 하지 마세요.
    * **배너:** 배너 이미지는 1920x576 해야 합니다. 이는 필요에 따라 자동으로 크기가 조정 되 고 이벤트 정보 또는 웹 페이지에 표시 됩니다. 반 투명 오버레이는 이미지의 하위 25%를 포함 합니다. 해당 영역에 텍스트를 배치 하지 마십시오.
    * 또 다른 (더 빠른) 옵션은 **타일 이미지를 배경 배너 이미지로 복사** 하는 것입니다. 그러면 타일 이미지도 배너 이미지로 사용 됩니다. Try:)을 제공 합니다. 잘 보이지 않는 경우 새 배너 이미지를 만들 수 있습니다.
2. **VR:** 이 섹션에는 AltspaceVR 앱 내에서 이벤트 중 가상 환경에 적용 되는 기능이 포함 되어 있습니다.
    * **기본 상황별 역할:** 녹색 물음표는 *이벤트의 모든 대상 그룹 멤버* 에 사용할 수 있는 특정 역할을 나열 합니다. 가장 일반적인 *megaphone_only* 입니다. 이를 추가 하 여 호스트 도구 단추 아래의 ' 강화 My Voice ' 옵션을 모든 대상 그룹 구성원에 게 제공 합니다. 이벤트에 비행이 필요한 경우 *파일럿* 을 추가 합니다. 둘 다 추가 하려면 *megaphone_only, 파일럿* 의 형식입니다. 각 사용자는 설정/Input/Fly.로 이동 하 여 AltspaceVR 앱에서 즉시 모드를 사용 하도록 설정 해야 합니다.
    * **지침:** [지침은](../world-building/adding-welcome-messages.md) 공간에 도착 했을 때 시작 이미지를 생성 하는 텍스트에 대 한 것입니다. 사용자가 [확인]을 선택 하 여 보기에서 제거 해야 합니다. 이는 "말하기에 초대 될 때까지 음소거 상태 유지" 또는 "ABC를 다룰 XYZ 이벤트 시작"과 같은 대상 그룹 지침을 제공 하는 데 주로 사용 됩니다. 이는 필요 하지 않으므로 신중 하 게 사용 합니다. Markdown를 사용 하 여 색 및 글꼴 크기 조정을 추가할 수도 있습니다. 자세한 내용은 다음과 같습니다. [http://digitalnativestudios.com/textmeshpro/docs/rich-text](http://digitalnativestudios.com/textmeshpro/docs/rich-text)

## <a name="event-page-actions"></a>이벤트 페이지 동작

이벤트 웹 페이지에서 다음과 같은 작업 및 옵션을 사용할 수 있습니다.

1. **편집** 을 선택 하 여 이벤트 설정을 변경 합니다.
2. **종료 이벤트** 를 선택 하 여 이벤트를 종료 합니다 (예를 들어 이벤트가 조기에 발생 하는 경우).
3. **초안으로 설정** 을 선택 하 여 이벤트를 활성이 아닌 초안으로 설정 합니다.
4. 가져온 전 세계에 대 한 업데이트를 수행한 경우 이벤트에 표시 되는 변경 내용에 대해 **다시 가져오기** 를 선택 해야 합니다. 이벤트의 공간을 다시 설정 해야:)
5. 다음 이벤트에 대해 이벤트를 복제 하려면 **이벤트 복제** 를 선택 합니다. 이 중복은 내 이벤트/내 초안 이벤트에 표시 되며, 새 날짜/시간을 업데이트 하 고 여기에서 활성화 해야 합니다.
6. **기본 이벤트로 추가** 를 선택 하 여 이벤트를 일정 목록으로 설정 합니다.
7. 이벤트를 완전히 삭제 하려면 **이벤트 삭제** 를 선택 합니다 (복구할 수 없음).

준비가 되 면 AltspaceVR 인터페이스에서 이벤트 **> 내 이벤트** 메뉴로 이동 하 여 공간을 다시 설정 하거나 변경 내용을 검사 하거나 사용자 지정을 계속 하려면 이벤트 공간을 입력 합니다. 

> [!IMPORTANT]
> 이는 복사본 이며 이벤트 공간에 대 한 사용자 지정은이 이벤트에만 해당 됩니다. 이벤트의 공간을 사용자 지정 하는 데 많은 시간을 소비 하는 경우 **반복적인 사용 이벤트** 에 대해 전 세계를 만드는 것이 더 좋을 수 있습니다.

## <a name="using-the-multimedia-console-for-a-slide-presentation"></a>슬라이드 프레젠테이션에 멀티미디어 콘솔 사용

## <a name="branding-your-event-with-images"></a>이미지를 사용 하 여 이벤트 브랜딩

이벤트 공간이 나 회사 로고 및 브랜딩에 이미지를 추가 하는 것은 물론, 사용자의 팀 이나 기타 시각적 콘텐츠가 AltspaceVR에서 쉽게 만들 수 있습니다.

이미지 요구 사항은 다음과 같습니다.
* JPG 또는 PNG 파일이 수락 됩니다.
* 최대 이미지 크기: 1920x1080
* 해결 방법: 72 DPI
* 파일 크기: 파일당 250KB 미만

이 프로세스에는 [AltspaceVR 사진](https://account.altvr.com/photos)이미지를 업로드한 다음, 세계 편집기 내의 사진 사용하여 이미지를 가져와서 이벤트 세계 공간에 배치하는 과정이 포함됩니다.

1. AltspaceVR 웹 사이트에서 [사진](https://account.altvr.com/photos) 이동합니다.
2. **업로드** 를 선택합니다.
3. **찾아보기를** 선택하여 컴퓨터에서 대화 상자를 열고 최적화된 이미지를 찾아 선택합니다.
4. **열기** 를 선택합니다.
5. **사진 만들기를** 선택합니다.
6. 모든 이미지에 대해 이 작업을 반복합니다.

AltspaceVR 및 이벤트 세계에서 이미지를 배치하려는 위치 근처에 자신을 배치합니다. 반복 **사용 이벤트에** 대해 이 이벤트를 만드는 경우 이벤트 공간이 아닌 세계에 있는지 확인합니다.

1. 사용자 인터페이스의 오른쪽 아래 모서리에서 **세계 편집기/편집기 패널을** 선택합니다.
2. 아래쪽에서 **마인** 탭을 클릭합니다.
3. **사진** 선택합니다.
4. 사진 업로드의 시간순으로 저장되므로 업로드된 최신 이미지가 목록의 첫 번째 이미지여야 합니다. 이 옵션을 선택하여 세계에 추가합니다.
5. 디바이스 커서를 사용하여 이미지를 잡고 원하는 위치에 놓습니다.
    * 컨트롤러를 사용하여 크기를 조정(일반적으로 VR의 터치 패드를 가로로 슬라이딩하거나 2D 모드에서 마우스를 사용).
    * 이미지의 위치와 회전을 미세 조정하려면 편집기에서 이미지의 **GEAR** 기호를 선택하고 회전 및 위치를 적절하게 설정합니다.
    * 크기 조정 기능을 통해 이미지의 크기를 조정할 수도 있습니다.

> [!NOTE]
> 편집 모드가 켜지면(주황색 텍스트) 자동으로 플라이할 수 있어 이벤트 세계에서 이미지를 더 높게 배치하기 위해 더 쉽게 이동할 수 있습니다.

6. 각 이미지 배치에 대해 이 작업을 반복합니다.
7. 이미지가 설정되면 개별적으로 잠그지 않은 경우 **모두 잠금을** 선택하고 편집 모드를 해제하고 편집기를 닫습니다.
8. 이벤트당 최대 5개의 사진 사용합니다.

## <a name="adding-3d-objects-using-world-editor-and-kits"></a>세계 편집기 및 키트를 사용하여 3D 개체 추가

AltspaceVR을 사용하면 세계 편집기를 통해 3D 개체를 세계로 가져올 수 있습니다. 현재 사용할 수 있는 많은 World Editors Kit가 있으며 그 과정에서 더 많은 것이 제공됩니다. 우주선, 로켓, 애니메이션 동물, 동물, 트리, 동물 및 다양한 개체가 있습니다. 또는 퍼블릭 컬렉션 또는 개인 용도로 사용자 고유의 키트를 추가하거나 [AltspaceVR MRE 앱](../world-building/using-mixed-reality-extensions.md)중 하나를 사용하는 것이 좋습니다. 이 앱은 상호 작용 가능한 개체(예: 귀사, 고양이, 앞, 의류, hat 또는 기타 상호 작용 가능한 개체)를 전 세계에 가져옵니다. 개체 목록이 곧 확장됩니다.

여기에 설명된 지침은 키트에 이미 있는 항목을 사용하는 데 초점을 맞춥니다. AltspaceVR에 3D 개체를 추가하고 가져오는 방법에 대한 자세한 내용은 ["glTF 모델을 내 세계로 가져오는 방법"을 참조하세요.](../world-building/importing-models.md) 및 ["어떻게 할까요? 내 키트를 업로드하세요?"](../world-building/uploading-custom-kits.md).

World Building Kits의 기존 3D 개체를 이벤트 세계에 추가하려면 세계 편집기/편집기 패널이 열려 있고 편집 모드가 활성화되어 있습니다.

1. 가져오려는 개체에 대한 키트를 탐색하고 선택합니다.
2. 커서가 있는 키트 개체를 선택하고 배치합니다. 세계 편집기에서 개체의 기어 아이콘을 사용하여 수동 번호 입력에 위치, 회전 및 배율 옵션을 사용합니다.
3. 개체가 올바른 위치에 배치되면 잠금 아이콘을 클릭하여 잠급 수 있습니다.
4. 원하는 경우 다른 개체를 추가합니다. Oculus Quest 사용자 지원을 향상하기 위해 3D 개체를 추가하고 5개의 개별 키트에서 선택한 항목을 더 이상 항목으로 유지하지 않도록 신중하게 해야 합니다. 한 세계에서 200개 이상의 개체를 사용하지 마세요.
5. 완료되면 모든 항목을 잠그고, 편집 모드를 해제하고, World Editor를 닫습니다. 돌아다니는. 몇 명의 테스터를 초대하여 방문하여 확인하고 멋진 이벤트를 준비하세요!
