# <a name="creating-an-event"></a>이벤트 만들기

AltspaceVR에서 이벤트를 만들기 위한 단계별 가이드입니다. AltspaceVR의 여러 이벤트에 참가하여 작동 방식을 파악하는 것이 좋습니다. [AltspaceVR 이벤트 달력에서](https://account.altvr.com/events) 모든 이벤트 목록을 확인합니다.

이 문서에서는 다음에 대해 알아봅니다.

* AltspaceVR 이벤트 용어
* 이벤트 및 이벤트 공간 또는 세계 만들기
* 슬라이드 프레젠테이션에 멀티미디어 콘솔 사용
* 이미지를 통해 이벤트 브랜딩
* 세계 편집기/키트를 사용하여 3D 개체 추가
* 내 이벤트를 기록하거나 라이브 스트림 방법

## <a name="altspacevr-event-terminology"></a>AltspaceVR 이벤트 용어

다음은 이벤트 및 이벤트 공간을 만드는 데 숙지해야 하는 용어입니다.

</br>

| 용어 | 정의 |
|---|---|
| 세계 빌드 | AltspaceVR은 가상 세계를 만들고 사용자 지정하는 기능을 제공합니다. AltspaceVR 호스트는 설명서, 디스패코드 채널 및 세계 내 이벤트를 지원하여 [도움말을 얻고 세계 빌드를 시작하는 방법에 대해 자세히 알아볼 수 있도록 합니다.](../world-building/world-building-faq.md) |
| World | 세계는 AltspaceVR의 가상 공간입니다. 사무실 또는 광역 범위일 수 있습니다. 사용자 지정이 가능한 환경입니다. 우주 내에 있으며 해당 우주 내에 여러 개의 세계도 있을 수 있습니다. 특수 이벤트, 학습 센터 및 다양한 회의 공간을 위한 여러 이벤트 공간을 준비하려는 경우 동일한 Universe 아래에 세계를 추가하여 함께 그룹으로 유지합니다. |
| 우주 | AltspaceVR 세계의 우주 건물 언어는 세계의 분류를 나타냅니다. 각 우주는 여러 세계를 보유할 수 있습니다. 세계는 우주의 설정을 상속하므로 여러 세계 및 기타 제어 기능에 대한 허용 목록에 사람을 쉽게 추가할 수 있습니다. 자세한 내용은 [세계 관리를 참조하세요.](../world-building/managing-worlds.md) |
| 템플릿 | 템플릿(또는 공간 템플릿)은 World Building 기능을 사용하여 템플릿을 만드는 대신 사용할 수 있는 미리 만들어진 세계 또는 환경입니다. AltspaceVR은 다양한 환경 및 이벤트에 대한 다양한 템플릿을 제공합니다. |
| 이벤트 공간 | 이벤트 공간은 AltspaceVR의 세계와 동의어입니다. 일반적으로 이벤트를 호스트하는 데 사용되는 세계를 나타냅니다. |
| 웹 사이트 | "웹 사이트"에 대한 참조는 [AltspaceVR의 웹 사이트 입니다.](https://altvr.com/) VR 디바이스가 아닌 컴퓨터 또는 태블릿의 [이벤트 웹 페이지를](https://account.altvr.com/events/my) 통해 이벤트를 만들고 편집하는 것이 더 쉬운 경우가 많습니다. [전 세계 빌드를](../world-building/managing-worlds.md) 위해 웹 사이트에도 액세스해야 합니다. |
| 상황별 역할 | [상황별 역할은](../getting-started/roles.md) 이벤트 작성자 또는 세계 작성기에서 할당됩니다. 이러한 역할은 세계 또는 이벤트 공간의 사용자에게 추가 기능과 기능을 제공합니다. 현재 이러한 구성은 호스트, 중재자, 파일럿(비행), Terraformer(세계 빌드) 및 Megaphone 전용으로 구성됩니다. 이러한 할당은 개별적으로 또는 전역적으로 할당될 수 있으며, 모든 사용자가 이벤트 공간 또는 세계에서 동일한 역할을 갖도록 할 수 있습니다. |
| 사용자 인터페이스(UI)/메뉴 | AltspaceVR In-world에 있는 경우 몰입형 환경 내에서 화면의 왼쪽과 오른쪽에 메뉴가 있습니다. AltspaceVR 로고가 있는 원 또는 주 메뉴는 기본 UI(사용자 인터페이스) 또는 메뉴를 열어 AltspaceVR을 탐색하고 환경을 사용자 지정하기 위한 다양한 화면에 액세스합니다. 선택적 UI 항목은 화면의 적절한 크기에 있으며 일반적으로 세계 편집기 및 호스트 도구를 포함합니다. 커서를 클릭하여 열고 상호 작용합니다. |
| SDK/MRE | 이러한 용어는 소프트웨어 개발 키트 및 [Mixed Reality 확장과](../world-building/using-mixed-reality-extensions.md) 관련된 세계 빌드 용어로, 전 세계에 기능과 기능을 추가하는 데 사용됩니다. 이러한 항목은 일반적으로 고급 사용자를 위한 것입니다. |

## <a name="understanding-events"></a>이벤트 이해

AltspaceVR을 사용하면 미리 만들어진 템플릿 공간을 사용하여 특정 요구 사항에 맞게 사용하거나 사용자 지정하거나 처음부터 세계를 만들 수 있는 세계를 쉽게 만들 수 있습니다. 이 문서에서는 미리 만들어진 템플릿을 사용하여 이벤트를 만드는 방법을 다룹니다. 이미지로 [이벤트 브랜딩](#branding-your-event-with-images) 및 세계 [편집기 및 키트를 사용하여 3D 개체 추가에](#adding-3d-objects-using-world-editor-and-kits) 대한 아래 섹션에는 추가 사용자 지정에 대한 팁이 포함되어 있습니다. 사용자 지정 세계를 처음부터 빌드하는 데 대한 자세한 내용은 AltspaceVR의 World Building Tour 모임에 참가하고 [World Building 지원 설명서를 확인하세요.](../world-building/world-editor-getting-started.md)

AltspaceVR은 이벤트에 대한 공간을 만드는 두 가지 방법을 제공합니다.

* **일회성 사용:** 이벤트를 만들고 템플릿 세계를 선택합니다.
* **반복 사용:** 세계 공간을 만들고 이벤트로 가져옵니다.

이벤트를 만드는 프로세스는 두 가지 모두에 대해 동일합니다. 한 가지 예외는 [세계를 만들고](../world-building/managing-worlds.md) 반복 사용 이벤트 공간으로 가져오는 것입니다. 또한 다음을 알아야 합니다.

</br>

| 용어 | 정의 |
|---|---|
| 세계 복사 | 반복되는 이벤트에 세계 공간을 사용하면 지속적인 사용자 지정이 허용됩니다. 서명, 이미지, 생성 지점 및 기타 사용자 지정은 매번 이벤트 공간을 사용자 지정하는 대신 이벤트에서 이벤트로 유지 관리됩니다. |
| 이벤트 세계 사용자 지정 | 이벤트가 만들어지면 템플릿 또는 가져온 World가 복사되어 해당 이벤트에 잠깁니다. 이벤트 세계를 변경하고 원래 세계와 그 반대로 영향을 미치지 않을 수 있습니다. 휴일 장식, 사진 또는 기타 장식용 일회성 사용 세계 편집기 요소를 추가하여 이벤트 공간을 이벤트에 더 늘늘이고 적절하게 만드는 것이 좋습니다. |

이벤트 공간이 업데이트되지 않는 한 원래 세계에 대한 수정은 이벤트 세계에 없습니다.
1. 이벤트 웹 페이지의 **RE-IMPORT WORLD** 단추를 사용합니다.
2. 동기화를 완료하는 데 2-3분이 걸립니다. 
3. 이벤트 공간이 변경되지 않은 경우 설정 **> 보통 > 공간 다시 설정으로** 이동하여 이벤트 공간을 다시 설정합니다. 세계 작성기로서 공간을 많이 다시 설정하겠습니다.

요소가 제대로 로드되지 않는 것과 같은 기술적인 문제가 있는 경우 AltspaceVR 메뉴로 이동하여 **설정 > 보통 > 공간 다시 설정을** 선택하여 이벤트 공간을 다시 설정하고 새 변경 내용을 확인합니다. 2D의 Windows PC 사용자는 AltspaceVR의 **Ctrl + Alt + R** 바로 가기 키를 사용하여 공간을 신속하게 다시 설정할 수 있습니다.

## <a name="creating-your-event-and-event-space-or-world"></a>이벤트 및 이벤트 공간 또는 세계 만들기

다음은 일회성 또는 반복 이벤트에 대한 이벤트를 만들기 위한 단계별 지침입니다. 템플릿을 선택하거나 이벤트에 대한 세계를 가져옵니다. 

> [!NOTE]
> AltspaceVR 이벤트 웹 페이지는 녹색 물음표 단추를 사용하여 양식의 모든 측면에 대한 지침을 제공합니다. 특정 지침을 위해 커서를 커서 위로 굴려 놓습니다.

AltspaceVR 웹 사이트의 [이벤트 > 내 이벤트](https://account.altvr.com/events/my) 웹 페이지에서 **이벤트 예약을** 선택하거나 [AltspaceVR 웹 페이지에서 이벤트 만들기로](https://account.altvr.com/events/new)바로 이동합니다.

1. **이벤트 제목:** 이벤트 이름을 입력합니다. AltspaceVR 웹 사이트 및 세계 내 인터페이스의 다양한 달력 보기에서 볼 수 있도록 구체적이고 간결하게 유지합니다. 단어 사이에 공백을 포함하여 제목을 23자 미만으로 유지합니다.
2. **설명:** 이벤트에 대한 설명을 입력합니다.
    * 설명에 10자 이상이어야 합니다. 그렇지 않은 경우 이벤트가 만들어지지 않습니다.
    * 단락 사이에 빈 공백을 추가합니다.
    * 이벤트의 Facebook, Discord, 웹 사이트 또는 기타 리소스에 대한 링크를 추가하려면 다음 형식을 사용합니다(이 Markdown은 웹 사이트의 이벤트 승격 페이지에서만 작동하며 AltspaceVR 메뉴에서 제대로 렌더링되지 않습니다.) `[Event Name](http://example.com/)`

3. **시작 날짜/종료 날짜:** 시작 시간을 설정하고 종료 시간이 시작 시간 이후인지 확인합니다.
4. **범주:** 이벤트 유형을 가장 잘 설명하는 범주를 선택합니다. 
5. 이벤트를 프라이빗 또는 **퍼블릭** 로 설정합니다. 
    * 표시 유형: 공용 이벤트는 AltspaceVR 웹 사이트 이벤트 달력의 [모두 탭에](https://account.altvr.com/events/all) 표시되며 공개적으로 열립니다.
    * 프라이빗 이벤트는 AltspaceVR 이벤트 달력에 표시되지 않으며 이벤트 공간을 입력하려면 이벤트 URL이 필요합니다.
    * '주 이벤트로 추가'하려면 이벤트를 public으로 설정해야 합니다.
6. **템플릿 선택:** 웹 페이지의 오른쪽에는 AltspaceVR에서 사용 가능한 템플릿의 미리 보기 이미지 목록이 있습니다. 게임 방, 사무실, 모임 공간, 프레젠테이션 공간 및 재미 있는 모임 공간이 있습니다. 흥미를 보이는 항목을 선택 합니다. 원하지 않는 경우 계속 진행 하 여 새 템플릿으로 새 이벤트를 만듭니다. 사용자 고유의 사용자 지정 이벤트 세계를 만들려면 기본 템플릿으로 **쿨 하늘** 을 선택 하 고 전 세계를 가져오는 다음 지침을 따릅니다.
7. **고급 옵션** 선택

### <a name="advanced-options"></a>고급 옵션

1. **수준 올리기:** 각 이벤트에는 두 개의 브랜딩 이미지가 필요 합니다. 이러한 이미지는 AltspaceVR 웹 사이트에 표시 되며 AltspaceVR 내부의 이벤트에 나타나지 않습니다.
    * **타일:** 타일 이미지는 1920 x 1080 픽셀 이어야 합니다. 이 이미지는 다양 한 축소판 그림 크기에 맞게 크기가 조정 되 고 AltspaceVR events 달력에서 다른 이벤트와 함께 표시 됩니다. 이미지가 명확 하 고 많은 텍스트를 포함 하지 않는지 확인 합니다. 소유 하지 않은 저작권 이미지/로고는 사용 하지 마세요.
    * **배너:** 배너 이미지는 1920x576 해야 합니다. 이는 필요에 따라 자동으로 크기가 조정 되 고 이벤트 정보 또는 웹 페이지에 표시 됩니다. 반 투명 오버레이는 이미지의 하위 25%를 포함 합니다. 해당 영역에 텍스트를 배치 하지 마십시오.
    * 또 다른 (더 빠른) 옵션은 **타일 이미지를 배경 배너 이미지로 복사** 하는 것입니다. 그러면 타일 이미지도 배너 이미지로 사용 됩니다. Try:)을 제공 합니다. 잘 보이지 않는 경우 새 배너 이미지를 만들 수 있습니다.
2. **VR:** 이 섹션에는 AltspaceVR 앱 내에서 이벤트 중 가상 환경에 적용 되는 기능이 포함 되어 있습니다.
    * **기본 상황별 역할:** 녹색 물음표는 *이벤트의 모든 대상 그룹 멤버* 에 사용할 수 있는 특정 역할을 나열 합니다. 가장 일반적인 *megaphone_only* 입니다. 이를 추가 하 여 호스트 도구 단추 아래의 ' 강화 My Voice ' 옵션을 모든 대상 그룹 구성원에 게 제공 합니다. 이벤트에 비행이 필요한 경우 *파일럿* 을 추가 합니다. 둘 다 추가 하려면 *megaphone_only, 파일럿* 의 형식입니다. 각 사용자는 설정/입력/날아오기로 이동 하 여 AltspaceVR 앱에서 즉시 모드를 사용 하도록 설정 해야 합니다.
    * **지침:** [지침은](../world-building/adding-welcome-messages.md) 공간에 도착 했을 때 시작 이미지를 생성 하는 텍스트에 대 한 것입니다. 사용자가 [확인]을 선택 하 여 보기에서 제거 해야 합니다. 이는 "말하기에 초대 될 때까지 음소거 상태 유지" 또는 "ABC를 다룰 XYZ 이벤트 시작"과 같은 대상 그룹 지침을 제공 하는 데 주로 사용 됩니다. 이는 필요 하지 않으므로 신중 하 게 사용 합니다. Markdown를 사용 하 여 색 및 글꼴 크기 조정을 추가할 수도 있습니다. 자세한 내용은 다음과 같습니다. [http://digitalnativestudios.com/textmeshpro/docs/rich-text](http://digitalnativestudios.com/textmeshpro/docs/rich-text)
3. **고급:** 다음은 이벤트의 고급 기능에 대 한 간략 한 설명입니다. 이벤트가 생성 된 후 이벤트를 편집할 때까지 이러한 기능 중 일부는 표시 되지 않습니다.

        * 이벤트를 삭제 합니다.
    * **그룹:** 드롭다운 메뉴를 사용 하 여 개인 [그룹](group-features.md) 에서 선택 합니다. 계정이 그룹에 생성/추가 된 경우에만 표시 됩니다.

    * **폰만 해당:** 호스트 도구 단추를 통해 해당 인터페이스에 강화 내 음성 단추를 추가 합니다.
    * **Terraformer:** 인터페이스에 세계 편집기 단추를 추가 합니다.
    * **파일럿:** 해당 개인의 비행 기능을 추가 합니다. 참고: 설정/입력/즉석에서이를 사용 하도록 설정 해야 합니다.
    * **음악 수행자:** 콘서트 및 악기 이벤트와 관련 된 기능 및 기능을 추가 합니다.
6. **차단 된 사용자:** 사용자의 액세스를 차단 하려는 경우 또는 사용자가 이전에 중재자 또는 호스트에 의해 이벤트에서 제거 되 고 해당 이벤트가 중복 되 면 나열 된 사용자 이름이 나열 됩니다. 이벤트 호스트나 관리자는 언제 든 지 차단 된 사용자를 추가 하거나 제거할 수 있습니다.

양식이 완성 되 고 삼중이 선택 되 면 **이벤트 만들기** 를 선택 합니다.

> 이벤트가 올바르게 생성 되지 않은 경우 설명에 10 자가 이상 있거나 템플릿을 선택 했는지 확인 합니다. 선택 하는 경우 템플릿 이름이 흰색에서 blue로 바뀝니다.

## <a name="event-page-actions"></a>이벤트 페이지 동작

이벤트 웹 페이지에서 다음과 같은 작업 및 옵션을 사용할 수 있습니다.

1. **편집** 을 선택 하 여 이벤트 설정을 변경 합니다.
2. **종료 이벤트** 를 선택 하 여 이벤트를 종료 합니다 (예를 들어 이벤트가 조기에 발생 하는 경우).
3. **초안으로 설정** 을 선택 하 여 이벤트를 활성이 아닌 초안으로 설정 합니다.
4. 가져온 전 세계에 대 한 업데이트를 수행한 경우 이벤트에 표시 되는 변경 내용에 대해 **다시 가져오기** 를 선택 해야 합니다. 이벤트의 공간을 다시 설정 해야:)
5. 다음 이벤트에 대해 이벤트를 복제 하려면 **이벤트 복제** 를 선택 합니다. 이 중복은 내 이벤트/내 초안 이벤트에 표시 되며, 새 날짜/시간을 업데이트 하 고 여기에서 활성화 해야 합니다.
6. **기본 이벤트로 추가** 를 선택 하 여 이벤트를 일정 목록으로 설정 합니다.
7. 이벤트를 완전히 삭제 하려면 **이벤트 삭제** 를 선택 합니다 (복구할 수 없음).

준비가 되 면 AltspaceVR 인터페이스에서 이벤트 **> 내 이벤트** 메뉴로 이동 하 여 공간을 다시 설정 하거나 변경 내용을 검사 하거나 사용자 지정을 계속 하려면 이벤트 공간을 입력 합니다. 

> [!IMPORTANT]
> 이는 복사본 이며 이벤트 공간에 대 한 사용자 지정은이 이벤트에만 해당 됩니다. 이벤트의 공간을 사용자 지정 하는 데 많은 시간을 소비 하는 경우 **반복적인 사용 이벤트** 에 대해 전 세계를 만드는 것이 더 좋을 수 있습니다.

## <a name="using-the-multimedia-console-for-a-slide-presentation"></a>슬라이드 프레젠테이션에 멀티미디어 콘솔 사용

멀티미디어 콘솔은 이미지, 오디오 또는 비디오를 포함 하 여 프레젠테이션을 위한 콘텐츠를 구성 하는 좋은 방법입니다. [멀티미디어 콘솔 지침](multimedia-console.md) 에 따라 이벤트에 대 한 설정을 가져옵니다.

## <a name="branding-your-event-with-images"></a>이미지를 사용 하 여 이벤트 브랜딩

이벤트 공간이 나 회사 로고 및 브랜딩에 이미지를 추가 하는 것은 물론, 사용자의 팀 이나 기타 시각적 콘텐츠가 AltspaceVR에서 쉽게 만들 수 있습니다.

이미지 요구 사항은 다음과 같습니다.
* JPG 또는 PNG 파일이 수락 됩니다.
* 최대 이미지 크기: 1920 x 1080
* 해상도: 72 DPI
* 파일 크기: 파일당 250 KB 미만

이 프로세스에서는 [AltspaceVR 사진](https://account.altvr.com/photos)에 이미지를 업로드 한 다음 전 세계 편집기의 사진을 사용 하 여이를 가져와 이벤트 세계 공간에 저장 합니다.

1. AltspaceVR 웹 사이트의 [사진](https://account.altvr.com/photos) 으로 이동 합니다.
2. **업로드** 를 선택합니다.
3. **찾아보기** 를 선택 하 여 컴퓨터에서 대화 상자를 열고 최적화 된 이미지를 찾아 선택 합니다.
4. **열기** 를 선택합니다.
5. **사진 만들기** 를 선택 합니다.
6. 모든 이미지에 대해이를 반복 합니다.

AltspaceVR 및 이벤트 세계에서 이미지를 배치할 위치 가까이에 배치 합니다. **반복 된 사용 이벤트** 에 대해이를 만드는 경우에는 이벤트 공간이 아니라 전 세계에 있는지 확인 하세요.

1. 사용자 인터페이스의 오른쪽 아래 모서리에서 **세계 편집기/편집기 패널** 을 선택 합니다.
2. 맨 아래에 있는 **광산** 탭을 클릭 합니다.
3. **사진** 을 선택 합니다.
4. 사진은 업로드 순서 대로 저장 되므로 업로드 된 최신 이미지는 목록의 첫 번째 이미지 여야 합니다. 이를 선택 하 여 전 세계에 추가 합니다.
5. 장치 커서를 사용 하 여 이미지를 잡고 원하는 위치에 넣습니다.
    * 컨트롤러를 사용 하 여 크기를 조정할 수 있습니다. 일반적으로는 VR의 터치 패드에서 가로를 이동 하거나 2D 모드에서 마우스를 사용 하 여 크기를 조정 합니다.
    * 이미지의 위치와 회전을 세밀 하 게 조정 하려면 편집기에서 이미지의 **기어** 기호를 선택 하 고 회전 및 위치를 적절 하 게 설정 합니다.
    * 크기 조정 기능을 사용 하 여 이미지의 크기를 조정할 수도 있습니다.

> [!NOTE]
> 편집 모드가 켜져 있으면 (주황색 텍스트) 자동으로 이동 하 여 이미지를 더 쉽게 이동 하 여 이벤트 세계에 이미지를 배치할 수 있습니다.

6. 각 이미지 배치에 대해이를 반복 합니다.
7. 이미지가 제자리에 설정 된 경우 개별적으로 잠그지 않은 경우 **모두 잠금** 을 선택 하 고 편집 모드를 해제 한 후 편집기를 닫습니다.
8. 이벤트 당 최대 5 개의 사진만 사용 합니다.

## <a name="adding-3d-objects-using-world-editor-and-kits"></a>세계 편집기 및 키트를 사용 하 여 3D 개체 추가

AltspaceVR를 사용 하면 세계 편집기를 사용 하 여 3D 개체를 가져올 수 있습니다. 현재 사용할 수 있는 많은 세계 편집기 키트가 있습니다. Spaceships, rockets, 애니메이션 동물, sculptures, 나무, 식물 및 다양 한 개체가 있습니다. 또는 공용 컬렉션 또는 사용자 고유의 개인 사용에 대 한 키트를 추가 하거나, [AltspaceVR MRE 앱](../world-building/using-mixed-reality-extensions.md)중 하나를 사용 하 여 interactable 개체를 사용자에 게 추가 하는 것이 좋습니다. 개체 목록이 곧 확장 될 것으로 간주 합니다.

여기의 지침은 키트에 이미 있는 항목을 사용 하는 데 중점을 둡니다. 3D 개체를 AltspaceVR에 추가 하 고 가져오는 방법에 대 한 자세한 내용은 ["내 세계에 글 제 tf 모델을 가져오는 방법"](../world-building/importing-models.md) 을 참조 하세요. ["사용자 키트를 업로드 어떻게 할까요??"를 선택](../world-building/uploading-custom-kits.md)합니다.

세계의 편집기/편집기 패널 열기 및 편집 모드를 사용 하 여 세계 건물 키트의 기존 3D 개체를 이벤트 세계에 추가 하려면 다음을 수행 합니다.

1. 가져오려는 개체에 대 한 키트를 탐색 하 고 선택 합니다.
2. 커서를 사용 하 여 키트 개체를 선택 하 고 위치를 설정 합니다. 수동 숫자 항목의 위치, 회전 및 크기 조정 옵션을 사용 하려면 세계 편집기의 개체에서 기어 아이콘을 사용 합니다.
3. 개체를 올바른 위치에 배치 하는 경우 잠금 아이콘을 클릭 하 여 잠급니다.
4. 원하는 경우 다른 개체를 추가 합니다. 3D 개체를 추가 하는 것이 적절 합니다. 5 개의 개별 키트에서 항목 수를 초과 하지 않도록 선택 항목을 유지 하는 것이 좋습니다. 전 세계에서 200 개 이상의 개체를 사용 하지 마세요.
5. 완료 되 면 모든 항목을 잠그고 편집 모드를 해제 한 다음 전 세계 편집기를 닫습니다. Wander. 몇 명의 테스터를 방문 하 여 확인 하 고 멋진 이벤트를 준비 해 보세요.
