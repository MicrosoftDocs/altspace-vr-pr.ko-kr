### <a name="the-slightly-longer-version"></a>약간 더 긴 버전

1. [https://obsproject.com/](https://obsproject.com/)로 이동합니다.
2. OBS를 다운로드 하려면 **Windows** 를 선택 합니다. 이 게시물은 **Obs v 22.0.2** 를 사용 합니다.

### <a name="setting-up-live-streaming-on-youtube"></a>YouTube에서 라이브 스트리밍 설정

실시간 스트림 (라이브 스트리밍)을 실시간으로 가져오거나 이후 라이브 스트림 이벤트 (새 라이브 이벤트)를 설정할 수 있습니다. ' 새로운 라이브 이벤트 ' 방식으로 설정 하는 것이 좋습니다.

1. 브라우저를 열고에 로그인 합니다. [https://www.youtube.com/](https://www.youtube.com/)
2. 계정 아이콘을 선택 하 고 오른쪽 위에 있는 드롭다운에서 Creator Studio를 선택 합니다.
3. 페이지 왼쪽의 라이브 스트리밍입니다.

' Stream now ' 메서드 변경 미리 보기 기본 정보 인코더 설정을 변경 해야 하는 경우 서버 URL을 기본 _ * * 스트림 이름/키로 유지 합니다. ' 표시 '를 클릭 하 고이 키를 복사 합니다 .이 키를 스트림 키에 붙여넣기를 클릭 합니다. 그런 다음 확인을 클릭 하 여 클릭 스트리밍 전환 YouTube로 이동 합니다.
실제 YouTube 라이브 스트림 비디오 페이지를 보려면 아래로 스크롤하고 공유 링크에 대 한 오른쪽 아래를 확인 해야 합니다.
이 항목을 복사 하 여 새 브라우저 탭에 붙여넣은 다음 비디오를 클릭 합니다. 목록에서 비디오를 클릭 하면 지금 라이브로 표시 되 고이를 클릭 하면 YouTube 라이브 스트림 페이지가 표시 됩니다.
이 URL은 라이브 스트림 링크 이며 모든 소셜 채널에서 공유할 수 있습니다:) 라이브 스트림을 중지 하려면 OBS에서 스트리밍 중지를 클릭 합니다. 그러면 YouTube에서 라이브 스트림이 종료 됩니다.

' Events ' 메서드- https://www.youtube.com/my_live_events ' 새 라이브 이벤트 ' 클릭 제목, 날짜, 시작 시간, 설명 및 태그 추가 – AltspaceVR 태그를 잊지 마세요:) 드롭다운 메뉴에서 공용을 선택 합니다. 사용자 지정 미리 보기를 추가 하려면 [이벤트 만들기]를 클릭 하 고 이미지 찾아보기 및 업로드를 클릭 합니다. (1280x720 가장 좋음) Select: 단일 사용 스트림 키 선택 인코더: 기타 인코더 복사 스트림 이름 (스트림 키)

이제 ' 조사식에서 보기 ' 페이지를 마우스 오른쪽 단추로 클릭 하 고 ' 새 탭에서 링크 열기 '를 클릭 합니다.

이는 YouTube 라이브 스트림 이벤트 링크입니다 .이 링크를 실제 이벤트 앞에서 소셜 방식으로 공유할 수 있습니다.

이제 OBS 파일>설정 스트림을 엽니다. 방금 복사한 스트림 키를 스트림 키 필드에 붙여 넣은 다음, 다시 YouTube로 다시 확인 하 고, ' 변경 내용 저장 '을 마우스 오른쪽 단추로 클릭 하 고 ' 새 탭에서 링크 열기 '를 클릭 한 다음 다시 OBS로 이동 하 여 ' 스트리밍 시작 '을 다시 클릭 합니다. 이제 ' 미리 보기 ' 단추가 파란색으로 표시 됩니다. [미리 보기] 단추를 클릭 하 여 라이브 이벤트를 미리 볼 지 여부를 묻는 메시지가 표시 되 면 확인 잠시 기다렸다가 ' 스트리밍 시작 ' 단추를 클릭 하 여 라이브 이벤트를 스트리밍할 지 여부를 묻는 메시지를 표시 합니다.

이제 살고 있습니다.

' 조사식에 보기 ' 페이지가 열려 있는 브라우저 탭으로 이동 하 여 비디오가 제대로 보이는지 확인 합니다. Windows 볼륨 믹서에서 음소거 할 때 브라우저에서 오디오를 꺼 었으 므로 오디오를 듣지 못합니다. 휴대폰에서 오디오를 확인 하거나 친구에 게 오디오를 확인 하도록 요청 하세요.

라이브 스트림이 완료 되 면 YouTube의 Live Control 대화방 페이지로 돌아갑니다.

대화 상자가 열리고 라이브 이벤트 스트리밍을 중지할지 여부를 확인 합니다.

OBS로 이동 하 고 스트리밍 중지도 클릭 합니다.
