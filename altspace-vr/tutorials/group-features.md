# <a name="using-the-groups-feature-to-create-private-events"></a>그룹 기능을 사용하여 프라이빗 이벤트 만들기

그룹 기능을 통해 호스트는 승인된 게스트 목록에 열려 있는 VR 공간을 만들 수 있습니다. 호스트는 해당 공간에 대한 액세스 권한을 부여하려는 AltspaceVR 사용자 이름/이메일을 수동으로 추가할 수 있습니다. 이 기능을 사용하여 이벤트, 회의 및 회의를 호스트하는 방법은 다음과 같습니다.

> [!NOTE]
> "그룹"에 액세스하려면 altvr.com 로그인하고, 웹 사이트에서 사용자 프로필을 편집하고, 초기 액세스를 선택하여 초기 액세스 기능을 옵트인해야 합니다. 또는 AltspaceVR 앱 내에서 설정 일반, 초기 액세스 프로그램에 참여합니다.

1. 브라우저에서 로 이동하여  [https://altvr.com](https://altvr.com)   계정에 로그인합니다. 
2. **추가** 풀다운 메뉴를 찾습니다.
3. **그룹** 선택

![더 많은 풀다운 메뉴가 강조 표시된 AltspaceVR 웹 보기](images/group-features-img-01.png)

4. 그룹 페이지에서 자신이 속해 있거나 관리자인 그룹 목록을 볼 수 있습니다. 

![AltspaceVR 웹 보기 그룹 페이지](images/group-features-img-02.png)

5. **만들기** 단추를 선택하여 새 그룹을 만듭니다.
6. 필드를 입력하여 그룹을 만듭니다.

![입력 필드가 있는 새 그룹 만들기 페이지](images/group-features-img-03.png)

> [!NOTE]
> 녹색 "?"을 마우스로 가리켜도 됩니다. 을 하여 필드가 있는 필드에 대한 추가 정보를 얻습니다.

7. 모든 참석자에게 이메일을 보내고 동일한 이메일 주소를 사용하여 AltspaceVR 계정을 만들도록 합니다. AltspaceVR 계정을 만들거나 [웹 사이트에서](https://account.altvr.com/users/sign_up)만들 수 있습니다.

## <a name="how-to-create-a-private-eventfrom-a-group"></a>그룹에서 프라이빗 이벤트를 만드는 방법 

1. 구성원 또는 관리자 페이지에서 액세스하려는 그룹을 선택합니다. 
2. **관리자 로서** 이전에 구성한 모든 옵션을 수정하여 그룹을 만들 수 있습니다.그룹 페이지의 "편집" 단추로 이동하여 액세스합니다.  
3. **그룹에 대한 다른 관리자 추가:**   편집할 때 다른 사용자를 관리자로 추가하여 그룹을 관리할 수 있습니다.  
    * 계정 전자 메일이 아닌 사용자의 AltspaceVR 사용자 이름이어야 합니다.  
4. **이벤트 만들기** 를 선택합니다.  

![만든 이벤트](images/group-features-img-04.png)

![내 이벤트 패널이 강조 표시된 AltspaceVR 앱](images/group-features-img-06.png)

## <a name="how-to-add-an-email-list-to-a-group"></a>그룹에 메일 목록을 추가하는 방법 

큰 그룹이 있고 대량으로 추가해야 하는 경우 먼저 그룹과 연결된 모든 메일 주소에 대한 전자 메일 주소 목록을 만들어야 합니다. Microsoft 계정 연결에 등록하는 경우 이 단계를 수행하려면 해당 Microsoft 계정 주소가 필요합니다.

### <a name="creating-a-list"></a>목록 만들기

스프레드시트 프로그램을 사용하여 모든 전자 메일 주소가 있는 단일 열을 만듭니다. 이 작업을 수행했으면 다음을 수행합니다.  

1. 그룹을 만들거나 편집합니다.  
2. 멤버 전자 메일 필드를 **찾습니다.**  
3. 파일 **선택** 옵션을 찾아 *.csv 파일에서 이메일을 가져옵니다.
4. 목록을 한 줄당 하나의 전자 메일로 가져와야 합니다.

> [!NOTE]
> 선택하는 경우 *.csv 파일에서 가져오는 대신 전자 메일을 복사하여 필드에 붙여넣을 수도 있습니다.

### <a name="importing-a-list"></a>목록 가져오기

일부 플랫폼에서는 전용 "전자 메일" 열이 있는 *.csv 파일(쉼표로 구분된 값)을 내보낼 수 있습니다.이 열을 사용하도록 선택할 수 있습니다.이 작업을 사용할 수 있게 되면 단계는 대부분 동일합니다.  

1. 그룹을 만들거나 편집합니다.  
2. 멤버 전자 메일 필드를 **찾습니다.**  
3. 파일 **선택** 옵션을 찾아 *.csv 파일에서 이메일을 가져옵니다.
4. 목록을 한 줄당 하나의 전자 메일로 가져와야 합니다.

> [!NOTE]
> 선택하는 경우 *.csv 파일에서 가져오는 대신 전자 메일을 복사하여 필드에 붙여넣을 수도 있습니다.

## <a name="faq"></a>FAQ 

### <a name="im-having-trouble-filling-out-the-fields-in-groups-what-do-they-mean"></a>그룹의 필드를 작성하는 데 문제가 있습니다. 무엇을 의미할까요?

(*녹색 "?"을 마우스로 가리켜도 됩니다.을 사용하여 필드가 있는 필드에 대한 추가 정보를 얻습니다.) 

* **이름**   – 모든 멤버에 표시할 그룹의 이름입니다.   
* **설명**   – 그룹의 용도를 설명하는 필드입니다. 모든 멤버에 표시됩니다.
* **사용자**   – 이 필드를 사용하면 그룹에 대한 그룹 구성원을 줄당 하나씩 사용자 이름 또는 전자 메일 주소로 나열할 수 있습니다.그룹 멤버를 정의하는 한 가지 방법입니다.
    * 줄당 사용자 이름 또는 전자 메일을 입력합니다.  
    * AltspaceVR에서 사용자 이름을 찾으려면 사용자가 "주 메뉴"를 열고 "Me" 섹션으로 이동하도록 합니다.괄호 안의 이름입니다.  
    * 웹 사이트에서 사용자 이름을 찾으려면 사용자가 "추가 -> 프로필" 옵션에서 "프로필"로 이동하도록 합니다. 프로필 페이지의 왼쪽 열에서 찾을 수 있습니다.  
    * 필드는 대/소문자 구분이므로 "username" 및 "Username"은 다른 것으로 간주됩니다.  
    * 이 필드는 유효한 계정과 연결된 사용자 이름과 전자 메일을 모두 허용합니다.줄당 하나의 항목이 있는지 확인합니다.동일한 사람이면 전자 메일이나 사용자 이름을 둘 다 입력할 필요가 없습니다. 둘 중 하나가 작동합니다.  
    * 계정은 양호해야 합니다.  
    * 브라우저 "Ctrl + F"의 찾기 옵션을 통해 목록을 검색할 수 있습니다.  
* **이미지**   – 그룹의 프로모션 배너 이미지로 사용됩니다.그룹 웹 페이지 및 VR에 표시됩니다.  
    * 이미지 충실도를 높이기 위해 1920 x1080을 권장합니다.  
    * JPEG 또는 PNG 기본 설정.  
    * 이미지는 16:9 가로 세로 비율이어야 합니다.  
    * 업로드에서 썸네일을 자동으로 생성하고 AltspaceVR 및 웹 사이트의 다양한 UI 화면에서 사용합니다.  
* **프로필 이미지**   – 그룹 프로필 이미지로 사용됩니다. 그룹 웹 페이지 및 VR에 표시됩니다.  
    * 이미지 충실도를 높이기 위해 512×512를 권장합니다.  
    * JPEG 또는 PNG 기본 설정.  
    * 업로드에서 썸네일을 자동으로 생성하고 AltspaceVR 및 웹 사이트의 다양한 UI 화면에서 사용합니다.  
* **환경**   선택 – 페이지의 오른쪽에 있는 사용할 수 있는 환경의 다양한 썸네일이 표시됩니다. 이 환경은 그룹의 기본 행아웃 세계 역할을 하며 사용자와 그룹 구성원을 위해 "Name" 필드> Commons의 "Worlds -> My Worlds -> <Name"에 표시됩니다.  
    * 자신만의 세계를 만들려면 [웹 사이트](https://account.altvr.com/worlds/my)를 통해 만들어야 합니다. 가능한 옵션으로 해당 세계를 볼 수 있습니다. 

> [!NOTE]
> 이 기능은 베타입니다. 그룹은 환경 선택 대화 상자에서 기존 세계를 활용하는 것이 좋습니다.

* "Commons"라는 단어는 항상 모든 그룹의 행아웃 세계에 접미사로 표시됩니다.  
* **멤버 전자 메일**   – 이 필드를 사용하면 AltspaceVR 계정과 연결된 전자 메일로 사용자를 나열할 수 있습니다.  **이 목록을 사용하면 위에서 설명한 "사용자" 목록이 재정의됩니다.**   그룹의 멤버 자격을 관리하는 데 권장되는 방법입니다.  
    * 이 필드를 작성할 때 연결된 전자 메일에 AltspaceVR 계정이 나열되어 있는지 확인하거나 그룹에 대한 액세스 권한이 손실될 수 있습니다. 계정을 Microsoft 계정 연결에 연결한 경우 사용된 주소가 Microsoft 계정 주소인지 확인해야 합니다.  
    * 주소를 필드에 붙여넣을 때 줄당 하나의 전자 메일이 있는지 확인합니다.  
    * 이 목록은 유효한 AltspaceVR 계정 없이 전자 메일을 허용합니다.AltspaceVR 사용자 이름이 확인되고 전자 메일 주소는 확인되지 않습니다. 그러나 해당 전자 메일을 사용하여 AltspaceVR 계정을 만드는 경우 연결된 사용자는 그룹에 액세스할 수 있습니다.  
    * *.csv 파일에서 목록을 가져올 수 있습니다. "파일 선택" 옵션을 선택하고 파일을 선택합니다.
    * 구성원을 조정하는 보다 명확한 방법이기 때문에 큰 그룹을 관리하는 방법으로 멤버 전자 메일을 사용하는 것이 좋습니다.그룹 관리자는 사용자/멤버에게 AltspaceVR 계정을 만들기 위해 결제 플랫폼에서 사용한 것과 동일한 정확한 전자 메일 주소를 사용해야 한다는 사실을 알려야 합니다.  
    * 사용자 목록을 가져오는 방법에 대한 지침은 그룹으로 사용자 가져오기 섹션을 참조하세요.   
* **자동 가져오기**   사용 – 자동 가져오기는 전자 메일 허용 목록에서 실행되는 반복 프로세스입니다. 이제 전자 메일에 연결된 유효한 AltspaceVR 계정이 있는지 여부를 조정하고 액세스 권한을 부여합니다.   
    * csv 파일 업로드를 사용하는 경우 사용하도록 설정하는 것이 좋습니다.
* **YouTube 비디오 ID**   – 웹 사이트의 그룹 방문 페이지에 YouTube 비디오를 포함할 수 있습니다.그룹 웹 사이트에만 해당되며 AltspaceVR 내부에는 표시되지 않습니다.
    * 전체 주소가 아닌 비디오 ID를 사용해야 합니다. 비디오 ID는 YouTube에서 "="(같음) 로그온 후에 나타나는 문자 집합입니다.  
* **Twitter 핸들**   – Twitter 계정의 최신 트윗을 웹 사이트의 그룹 방문 페이지에 포함합니다.그룹 웹 사이트에만 해당되며 AltspaceVR 내부에는 표시되지 않습니다.
* **그룹**   만들기 – 페이지 아래쪽에 있는 단추입니다. 위에서 설명한 필드를 완료했으면 이 단추를 선택하여 그룹을 만듭니다.  

### <a name="im-having-trouble-managing-my-groups-what-do-i-do"></a>내 그룹을 관리하는 데 문제가 있습니다. 어떻게 해야 합니까?

그룹의 구성원 또는 관리자인 경우 이벤트를 관리하는 추가 도구가 있습니다. 이러한 옵션에 액세스하려면 [웹 사이트의](https://account.altvr.com/groups)그룹 페이지로 이동합니다.  

1. 위쪽 표시줄에서 구성원 또는 관리자를 선택하고 관리/업데이트하려는 그룹을 선택합니다. 
2. 구성원 또는 관리자 페이지에서 타일 이미지를 클릭하여 액세스하려는 그룹을 선택합니다.

![구성원 또는 관리 패널 열기](images/group-features-img-08.png)

모든 그룹 멤버 자격을 구성원 또는 관리자로 볼 수 있는 기본 페이지입니다.특정 옵션을 보려면 그룹을 선택합니다.

![그룹 멤버 자격 화면](images/group-features-img-09.png)

그룹의 멤버인 경우 표시되는 그룹 페이지입니다.

**멤버로:**

* 웹 사이트의 그룹 방문 페이지에 액세스하여 그룹에 대한 업데이트를 볼 수 있습니다.* 그룹을 "나가기" 단추가 있습니다.이렇게 하면 다음과 같은 결과가 발생합니다.  
    * 더 이상 그룹의 방문 페이지에 액세스할 수 없습니다.  
    * 그룹 "Commons" 행아웃 세계는 더 이상 보거나 입력하지 않습니다.  
    * 그룹 이벤트를 볼 수 없습니다.

![그룹 이벤트 페이지](images/group-features-img-10.png)

그룹의 관리자인 경우 표시되는 그룹 페이지입니다.그룹의 페이지를 입력하는 방법에 관계없이 관리자 옵션이 표시됩니다.

**관리자 권한:**   멤버와 동일한 옵션이 있지만 추가 기능이 있습니다.  

* 이전에 구성한 모든 옵션을 수정하여 그룹을 만들 수 있습니다.그룹 페이지의 "편집" 단추로 이동하여 액세스합니다.  
    * **그룹에 대한 다른 관리자 추가:**   편집할 때 다른 사용자를 관리자로 추가하여 그룹을 관리할 수 있는 옵션이 제공됩니다.  

> [!NOTE]
> 계정 전자 메일이 아닌 사용자의 AltspaceVR 사용자 이름이어야 합니다.  

* **또한 "이벤트 만들기"** 단추에 액세스할 수 있습니다.  
    * **그룹 페이지에서 이벤트를 만들면 이벤트에 대한 그룹 권한이 자동으로 설정됩니다.**  
    * 웹 사이트를 통해 이벤트를 만들기 위한 자세한 설명서는 [여기에 있습니다.](https://help.altvr.com/hc/en-us/articles/360033140854-Creating-an-Event)
* **"삭제" 단추를 눌러 그룹을 제거할 수도 있습니다.**

> [!NOTE]
> 이는 파괴적 작업입니다. 복원하려면 그룹을 다시 만들어야 합니다.

* 이는 그룹 소유자만 사용할 수 있습니다. 즉, 원래 그룹을 만든 사용자입니다.
* 그룹의 관리자는 사용할 수 없습니다.

> [!IMPORTANT]
