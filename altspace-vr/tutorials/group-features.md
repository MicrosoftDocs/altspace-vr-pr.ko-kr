# <a name="using-the-groups-feature-to-create-private-events"></a>그룹 기능을 사용 하 여 개인 이벤트 만들기

그룹 기능을 사용 하면 승인 된 게스트 목록에 열리는 VR 공간을 만들 수 있습니다. 호스트는 해당 공간에 대 한 액세스 권한을 부여 하려는 AltspaceVR 사용자 이름/메일을 수동으로 추가할 수 있습니다. 이 기능을 사용 하 여 이벤트, 회의 및 회의를 호스트 하는 방법은 다음과 같습니다.

> [!NOTE]
> "그룹"에 액세스 하려면 먼저 altvr.com에 로그인 하 고, 웹 사이트에서 사용자 프로필을 편집 하 고, 초기 액세스를 선택 하 여 기능에 대 한 초기 액세스를 옵트인 (opt in) 해야 합니다. 또는 AltspaceVR 앱 (설정, 일반) 내에서 초기 액세스 프로그램에 참여 합니다.

1. 브라우저에서로 이동 하 여  [https://altvr.com](https://altvr.com)   계정에 로그인 합니다. 
2. **더 많은** 풀 다운 메뉴를 찾습니다.
3. **그룹** 선택

![추가 풀 다운 메뉴가 강조 표시 된 AltspaceVR 웹 보기](images/group-features-img-01.png)

4. 그룹 페이지에서, 또는 관리자의 일부인 그룹 목록을 볼 수 있습니다. 

![AltspaceVR 웹 보기 그룹 페이지](images/group-features-img-02.png)

5. **만들기** 단추를 선택 하 여 새 그룹을 만듭니다.
6. 그룹을 만들기 위해 필드를 채웁니다.

![입력 필드가 있는 새 그룹 만들기 페이지](images/group-features-img-03.png)

> [!NOTE]
> 녹색 "?"를 마우스로 가리킬 수 있습니다. 필드를 포함 하는 필드에 대 한 추가 정보를 가져올 수 있습니다.

7. 모든 참석자에 게 전자 메일을 보내고 동일한 메일 주소를 사용 하 여 AltspaceVR 계정을 만들도록 합니다. AltspaceVR 또는 [웹 사이트](https://account.altvr.com/users/sign_up)에서 계정을 만들 수 있습니다.

## <a name="how-to-create-a-private-eventfrom-a-group"></a>그룹에서 개인 이벤트를 만드는 방법 

1. 구성원 또는 관리 페이지에서 액세스 하려는 그룹을 선택 합니다. 
2. **관리자는** 이전에 구성한 모든 옵션을 수정 하 여 그룹을 만들 수 있습니다.그룹 페이지에서 "편집" 단추로 이동 하 여 액세스 합니다.  
3. **그룹에 대 한 다른 관리자 추가:**   편집 하는 경우 그룹을 관리 하는 데 도움이 되는 다른 사용자를 관리자로 추가할 수 있습니다.  
    * 사용자의 AltspaceVR 사용자 이름 이어야 하며 계정 전자 메일은 아닙니다.  
4. **이벤트 만들기** 를 선택 합니다.  

![만든 이벤트](images/group-features-img-04.png)

## <a name="how-to-use-patreon-and-eventbrite-to-host-a-ticketed-vr-event"></a>Patreon 및 Eventbrite를 사용 하 여 Ticketed VR 이벤트를 호스트 하는 방법: 

Patreon, Eventbrite를 사용 하 여 Altspace에 ticketed 이벤트를 만들 수 있습니다.사용자의 전자 메일 주소 목록을 만들거나 지불 플랫폼에서 해당 정보를 내보내야 합니다.  

### <a name="creating-a-list"></a>목록 만들기

스프레드시트 프로그램을 사용 하 여 모든 전자 메일 주소를 포함 하는 단일 열을 만듭니다. 이 작업을 완료 한 후에는 다음을 수행 합니다.  

1. 그룹을 만들거나 편집 합니다.  
2. **멤버 전자 메일** 필드를 찾습니다.  
3. * .Csv 파일에서 전자 메일을 가져오려면 **파일 선택** 옵션을 찾습니다.
4. 목록은 줄당 하나의 전자 메일을 사용 하 여 가져와야 합니다.

> [!NOTE]
> 을 선택 하는 경우 * .csv 파일에서 가져오는 대신 전자 메일을 복사 하 여 필드에 붙여 넣을 수도 있습니다.

### <a name="importing-a-list"></a>목록 가져오기

Patreon 및 Eventbrite와 같은 플랫폼을 사용 하면 전용 "전자 메일" 열을 포함 하는 * .csv 파일 (쉼표로 구분 된 값)을 내보낼 수 있습니다.이 열을 선택 하 여 사용할 수 있습니다.이 작업을 사용할 수 있게 되 면 단계는 대체로 동일 합니다.  

1. 그룹을 만들거나 편집 합니다.  
2. **멤버 전자 메일** 필드를 찾습니다.  
3. * .Csv 파일에서 전자 메일을 가져오려면 **파일 선택** 옵션을 찾습니다.
4. 목록은 줄당 하나의 전자 메일을 사용 하 여 가져와야 합니다.

> [!NOTE]
> 을 선택 하는 경우 * .csv 파일에서 가져오는 대신 전자 메일을 복사 하 여 필드에 붙여 넣을 수도 있습니다.

### <a name="im-having-trouble-managing-my-groups-what-do-i-do"></a>내 그룹을 관리 하는 데 문제가 있습니다. 어떻게 해야 합니까?

그룹의 멤버 또는 관리자 인 경우 이벤트를 관리할 수 있는 추가 도구가 있습니다. 이러한 옵션에 액세스 하려면 [웹 사이트](https://account.altvr.com/groups)의 그룹 페이지로 이동 합니다.  

1. 위쪽 표시줄에서 구성원 또는 관리자를 선택 하 고 관리/업데이트 하려는 그룹을 선택 합니다. 
2. 구성원 또는 관리 페이지에서 타일 이미지를 클릭 하 여 액세스 하려는 그룹을 선택 합니다.

![멤버 또는 관리 패널 열기](images/group-features-img-08.png)

구성원 또는 관리자로 모든 그룹 멤버 자격을 확인 하는 기본 페이지입니다.특정 옵션을 보려면 그룹을 선택 합니다.

![그룹 멤버 자격 화면](images/group-features-img-09.png)

그룹 구성원 인 경우 표시 되는 그룹 페이지입니다.

**구성원으로:**

* 웹 사이트의 그룹의 방문 페이지에 액세스 하 여 그룹에 대 한 업데이트를 볼 수 있습니다. * 그룹을 "탈퇴" 하는 단추를 사용할 수 있습니다.이렇게 하면 다음과 같은 결과가 나타납니다.  
    * 더 이상 그룹의 방문 페이지에 대 한 액세스 권한이 없습니다.  
    * "Commons" hangout 세계 그룹을 더 이상 보거나 입력 하지 않습니다.  
    * 그룹 이벤트는 볼 수 없습니다.

![그룹 이벤트 페이지](images/group-features-img-10.png)

그룹의 관리자 인 경우 표시 되는 그룹 페이지입니다.그룹 페이지를 입력 하는 방법에 관계 없이 관리 옵션이 표시 됩니다.

**관리자 권한:**   멤버와 동일한 옵션을 갖지만 추가 기능을 사용할 수 있습니다.  

* 이전에 구성한 모든 옵션을 수정 하 여 그룹을 만들 수 있습니다.그룹 페이지에서 "편집" 단추로 이동 하 여 액세스 합니다.  
    * **그룹에 대 한 다른 관리자 추가:**   이제 편집할 때 그룹을 관리 하는 데 도움이 되는 다른 사용자를 관리자로 추가 하는 옵션이 있습니다.  

> [!NOTE]
> 사용자의 AltspaceVR 사용자 이름 이어야 하며 계정 전자 메일은 아닙니다.  

* **"이벤트 만들기"** 단추에 액세스할 수도 있습니다.  
    * **그룹의 페이지에서 이벤트를 만들면 해당 이벤트에 대 한 그룹 권한이 자동으로 설정 됩니다.**  
    * 웹 사이트를 통해 이벤트를 만드는 방법에 대 한 자세한 내용은 [여기](https://help.altvr.com/hc/en-us/articles/360033140854-Creating-an-Event)를 참조 하세요.
* **"삭제" 단추를 눌러 그룹을 제거할 수도 있습니다.**

> [!NOTE]
> 이는 파괴적인 작업 이므로 복원 하려는 경우 그룹을 다시 만들어야 합니다.

* 그룹 소유자만 사용할 수 있습니다. 즉, 원래 그룹을 만든 사용자만 사용할 수 있습니다.
* 그룹의 관리자는 사용할 수 없습니다.

### <a name="i-want-to-use-eventbrite-and-patreon-but-i-dont-see-where-to-get-the-email-addresses-for-my-groups"></a>Eventbrite 및 Patreon를 사용 하려고 하지만 그룹의 전자 메일 주소를 가져올 위치가 표시 되지 않습니다.

**Patreon ** 

[Patreon](https://www.patreon.com/)   는 creatives가 해당 콘텐츠에 대 한 구독 프로그램을 만들 수 있는 멤버 자격 플랫폼입니다.Patreon는 여러 가지 방법으로 pledge 데이터를 내보낼 수 있지만 CSV 방향에 집중 하겠습니다.  [여기](https://www.patreon.com/portal/how-to/export-pledge-data)에서이 정보에 대 한 도움말 페이지를 찾거나   아래 지침을 따를 수 있습니다.  

1. Patreon 사이트의 왼쪽에 있는 작성자 쪽 표시줄에서 "패 트 론" 필드를 찾습니다.  
2. 확장 하 여 "관계 관리자"를 찾아 선택 합니다.  
3. 페이지 맨 위에는 CSV 라는 다운로드 단추가 있습니다. 이 폴더를 선택합니다.  
4. Windows 대화 상자 필드를 팝 하 여 파일을 저장 합니다.저장합니다.  
5. AltspaceVR 그룹 페이지의 가져오기 지침에 따라이 파일을 사용 합니다.  

**Eventbrite ** 

[Eventbrite](https://www.eventbrite.com)   는 사용자가 이벤트에 대 한 티켓을 관리 하 고 만들 수 있도록 하는 티켓 웹 사이트입니다.Patreon와 마찬가지로 참석자 전자 메일 목록을 생성 하는 방법에는 여러 가지가 있습니다.CSV에 대 한 지침을 다시 살펴보겠습니다.  [여기](https://www.eventbrite.com/support/articles/en_US/How_To/how-to-edit-and-export-an-attendee-summary-report?lg=en_US)에서이 정보에 대 한 도움말 페이지를 찾거나   ,이 데이터를 추출 하는 한 가지 방법으로 아래 지침을 따를 수 있습니다.  

1. 프로필 메뉴에서 "이벤트 관리" 메뉴를 찾습니다. 
2. 참석자 전자 메일 목록을 가져오려는 이벤트를 선택 합니다. 
3. 이벤트의 대시보드에서 "Common Reports" 라는 범주를 찾습니다. 
4. "분석-> 이벤트 보고서"의 왼쪽 열 메뉴에서 "일반 보고서"를 찾을 수도 있습니다.   
5. "참석자 요약"을 선택 합니다. 
6. "보고서 결과" 섹션을 찾습니다. 
7. "CSV" 단추를 선택 하 여 파일을 다운로드 합니다.

> 베타 기능입니다. AltspaceVR는 무료 환경 이며, 참가자가 ticketed 이벤트에 액세스 하지 못하게 할 수 있는 환경에서 발생할 수 있는 기술적 문제를 담당 하지 않습니다.
